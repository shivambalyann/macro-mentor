# Macro Mentor — Single Page App

## How to run
1. Clone or download the project folder.
2. Open `index.html` in a browser (Chrome/Edge/Firefox).
   - No server required for the demo.
3. Fill inputs and click **Generate Plan**.
4. Use **Export CSV** to download the meal plan.

## Files
- `index.html` — main page
- `css/styles.css` — styling
- `js/app.js` — logic, BMR, macros, meal-plan generator, and chart rendering
- `data/foods.csv` — optional external food DB
